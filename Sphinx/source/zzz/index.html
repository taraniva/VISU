Welcome to VISU's documentation!¶

This project is intended for visualisation of unstructured meshes used in Computational Fluid Dynamics.

User guide¶

This section contains basic information regarding the usage of this module.

visu_func.connect_nodes(figure_nodes, first_node_ID, second_node_ID, nodes)¶

Plots a line between two selected nodes.

Parameters:
  • figure_nodes -- Matplotlib figure instance
  • first_node_ID -- integer ID of first node
  • first_node_ID -- integer ID of second node
  • nodes -- list of nodes
visu_func.draw(filename_mesh, filename_cellsid, filename_connectivity, filename_vals, val_name, x_min, x_max, y_min, y_max)¶

Main plotting routine. Generates nodal plot, coloured plot and mesh plot and saves them in .eps format.

Parameters:
  • filename_mesh -- Path to file containing nodal coordinates
  • filename_cellsid -- Path to file containing cell characteristics
  • filename_connectivity -- Path to file containing cell connectivity
  • filename_vals -- Path to file containing cell-centred variables
  • val_name -- Varible for coloured plot, either density, pressure or internal_energy
  • x_min -- "left" x-axis boundary for plotting
  • x_max -- "right" x-axis boundary for plotting
  • y_min -- "lower" y-axis boundary for plotting
  • y_max -- "upper" y-axis boundary for plotting

visu_func.insert_cell_variables(nc, cells, data_variables, data_cell_vertices)¶

Transfers cell variables from imported data tables into the cells.

Parameters:
  • nc -- integer representing the total number of cells
  • cells -- list containing all cell objects
  • data_variables -- pandas table containing all cell variables ordered by cell ID
  • data_cell_vertices -- pandas table containing list of vertices in all cells
visu_func.truncate_colourmap(cmap, minval, maxval, n=100)¶

Modifies the span of inserted colourmap.

Parameters:
  • cmap -- Matplotlib colormap instance. Example colourmaps eg. 'jet', 'viridis' etc.
  • minval -- float in interval [0,1] representing lower boundary colour
  • maxval -- float in interval [0,1] representing upper boundary colour
Returns:

Colourmap with updated colour span.
Return type:

Matplotlib colourmap instance.
